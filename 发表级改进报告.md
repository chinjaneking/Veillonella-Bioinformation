## 🔬 Methods段落模板

```markdown
### Genome Selection and Quality Control
We collected 31 Veillonella genomes from NCBI RefSeq (as of March 2026).
Genome quality was assessed based on assembly statistics (genome size,
N50, contig number, and GC content). All genomes passed quality control
with genome sizes ranging from 2.04 to 2.21 Mb.

### Pan-genome Analysis
Panaroo v1.5.x was used with sensitive mode to cluster gene families.
The pan-genome openness was assessed using the power-law model, yielding
γ = 0.3386, indicating a moderately open pan-genome.

### Phylogenetic Analysis
IQ-TREE v3.0.1 was used with ModelFinder (MFP) and 1000 ultrafast
bootstrap replicates to construct a maximum-likelihood tree based on
1,125 core gene alignments.
```
