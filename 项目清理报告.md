## 建议后续操作

### 可选清理

如需进一步释放空间，可考虑：

1. **删除 Docker 镜像**（回收 6.1 GB）
   ```bash
   docker rmi veillonella-analysis:latest
   ```
   ⚠️ 注意：删除后需要重新构建镜像才能运行分析

2. **压缩结果文件**
   ```bash
   tar -czvf results_backup.tar.gz results/
   rm -rf results/
   ```

3. **删除原始基因组**（如已上传至云端）
   ```bash
   rm -rf data/genomes/*.fna
   ```

### 建议保留

- **Docker 镜像**：便于重新运行分析
