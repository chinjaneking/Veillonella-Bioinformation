## 🎯 改进路线图

### 第一阶段：数据基础强化（必需）

#### 1. 扩展样本量
```yaml
# config_paper.yaml
max_genomes: 100  # 从16扩展到100
assembly_level: "all"  # 包含draft基因组
```

**理由**:
- 16株菌不足以支撑可靠的泛基因组分析
- 100株菌可以更好地估计泛基因组开放/闭合性质
- 符合Frontiers in Microbiology等期刊标准

#### 2. 添加质量控制 (CheckM2)
**新增工具**: CheckM2 v1.0.0

**功能**:
- 评估基因组完整性 (Completeness)
- 评估基因组污染 (Contamination)
- 过滤低质量基因组（<90%完整或>5%污染）

**输出**:
- Table S1中的质量列
- QC散点图（完整性vs污染）

#### 3. 添加ANI分析 (fastANI)
**新增工具**: fastANI v1.3.3

**功能**:
- 两两ANI计算
- 物种聚类（ANI > 95%为同种）
- ANI热图

**科学意义**:
- 确定物种边界
- 验证系统发育树
- 为分支颜色分组提供依据

#### 4. 泛基因组统计升级
**新增脚本**: `pangenome_statistics.py`

**功能**:
- 幂律模型拟合: `y = k * n^γ`
- 开放/闭合泛基因组判定（γ > 0.5为开放）
- 核心基因稳定曲线
- Heaps定律分析

**输出**:
- Figure 2: 泛基因组积累曲线（期刊级）
- 统计参数（用于Methods部分）

## 🚀 快速开始

### 1. 构建新Docker镜像
```bash
docker build -f Dockerfile_paper -t veillonella-paper:latest .
```

### 2. 运行完整分析
```bash
docker run -v $(pwd):/app veillonella-paper:latest \
    -s Snakefile_paper \
    --cores 8 \
    all
```

### 3. 分步运行（推荐）
```bash
# Step 1: 下载和QC
docker run -v $(pwd):/app veillonella-paper:latest \
    -s Snakefile_paper \
    qc_checkm2 ani_analysis

# Step 2: 注释和泛基因组
docker run -v $(pwd):/app veillonella-paper:latest \
    -s Snakefile_paper \
    prokka_annotate panaroo

# Step 3: 统计和可视化
docker run -v $(pwd):/app veillonella-paper:latest \
    -s Snakefile_paper \
    pangenome_statistics annotate_tree

# Step 4: 生成补充材料
docker run -v $(pwd):/app veillonella-paper:latest \
    -s Snakefile_paper \
    supplementary_materials
```

## 📝 论文撰写建议

### Methods部分模板
```
Genome Selection and Quality Control
We downloaded 100 Veillonella genomes from NCBI RefSeq (as of March 2026).
Genome quality was assessed using CheckM2 v1.0.0, filtering for
completeness ≥90% and contamination ≤5%.

Species Delineation
Average Nucleotide Identity (ANI) was calculated using fastANI v1.3.3.
Species clusters were defined at ≥95% ANI threshold.

Pan-genome Analysis
Panaroo v1.5.x was used with sensitive mode. Pan-genome openness
was assessed using the power-law model (γ = X.XX).

Phylogenetic Analysis
IQ-TREE v3.0.1 was used with ModelFinder (MFP) and 1000 ultrafast
bootstrap replicates.
```
